## Use of SciConstant 

1. Download  the [SciConstant.mat](https://github.com/place-bo/ScientificConstant_Matlab)
2. Add it into your working fold 
3. Load and use. 
``` matlab
load("SciConstant.mat");
SciConstant('c').value
SciConstant('c').unit
SciConstant('c').name
```
